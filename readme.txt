== Description ==

Auto Post Generator is a powerful WordPress plugin that automates the creation and publishing of SEO-optimized blog posts using OpenAI's GPT-4 API. It offers customizable prompts, automatic scheduling, and word count control to meet your blogging needs.

Key Features:
- Generate SEO-friendly content automatically.
- Customize the prompt to guide the content generation.
- Schedule posts in advance with future publication dates.
- Set post status as draft, published, or scheduled.
- Integrate OpenAI's GPT-4 API for advanced content generation.
- Control the word count of generated posts.
- Tag and categorize posts automatically.
- Schema FAQ generation included in every post for enhanced SEO.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/auto-post-generator` directory or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the "Auto Post Generator Pro" settings page in the WordPress dashboard and enter your OpenAI API key.
4. Customize your prompts, set your default category, and adjust other settings to your liking.
5. Use the "Create post now" option to generate a post instantly or schedule posts for future dates.

== Frequently Asked Questions ==

= Do I need an OpenAI API key to use this plugin? =
Yes, you need to have an active OpenAI API key to generate content using the GPT-4 model.

= Can I customize the generated content? =
Yes, you can customize the prompt used to generate the content, control the word count, and add your desired tags and categories.

= Is there a limit to the number of posts I can generate? =
The plugin allows you to generate an unlimited number of posts, but OpenAI's API usage will depend on your API plan and rate limits.
