== Descripción ==

Auto Post Generator Pro es el plugin más avanzado de WordPress para la creación automatizada de contenido. Soporta múltiples proveedores de IA (OpenAI y DeepSeek), ofrece amplias opciones de personalización, programación automática e incluye un potente generador de ideas para posts.

🚀 **NUEVO en la Versión 3.0:**
- **Integración DeepSeek AI**: Soporte completo para los modelos avanzados de IA de DeepSeek
- **Personalización Avanzada de Contenido**: Control detallado sobre estilo de escritura, tono y audiencia
- **Programación Automática**: Configura posts recurrentes con frecuencia personalizable
- **Generador de Ideas**: Genera ideas creativas para posts sobre cualquier tema
- **Interfaz Mejorada**: Pestañas organizadas para una mejor experiencia de usuario
- **Parámetros Avanzados de IA**: Ajusta temperatura, tokens y otros parámetros de IA

🔥 **Características Principales:**

== Instalación ==

1. Sube los archivos del plugin al directorio `/wp-content/plugins/auto-post-generator` o instálalo directamente desde la pantalla de plugins de WordPress.
2. Activa el plugin a través de la pantalla 'Plugins' en WordPress.
3. Ve a "Auto Post Generator Pro" en tu panel de WordPress.
4. Configura tu proveedor de IA y clave API en la pestaña "General".
5. Personaliza la configuración de contenido en la pestaña "Contenido".
6. Configura la programación en la pestaña "Programación" (opcional).
7. ¡Comienza a generar posts manualmente o deja que la automatización se encargue!

== Preguntas Frecuentes ==

= ¿Qué proveedores de IA son compatibles? =
El plugin soporta OpenAI (GPT-4, GPT-4 Turbo, GPT-3.5 Turbo) y DeepSeek (DeepSeek Chat, DeepSeek Coder). Puedes cambiar entre proveedores fácilmente.

= ¿Necesito claves API para ambos proveedores? =
No, solo necesitas una clave API para el proveedor que quieras usar. Puedes obtener claves de OpenAI en https://platform.openai.com/api-keys y claves de DeepSeek en https://platform.deepseek.com/api_keys.

= ¿Puedo programar posts automáticamente? =
¡Sí! El plugin incluye un potente sistema de programación que puede generar y publicar posts automáticamente en la frecuencia que desees (diaria, semanal, quincenal o mensual).

= ¿Cómo funciona el generador de ideas para posts? =
El generador de ideas usa IA para crear ideas relevantes basadas en tu tema y preferencias de tipo de contenido. Puede generar 5-20 ideas para tutoriales, listas, comparaciones, noticias o contenido general.

= ¿Puedo personalizar el estilo de escritura? =
¡Absolutamente! Puedes configurar el estilo de escritura, audiencia objetivo, tono, enfoque SEO y agregar instrucciones personalizadas para que el contenido coincida con la voz de tu marca.

= ¿Hay límite en los posts que puedo generar? =
El plugin en sí no tiene límites. El uso depende de los límites de API de tu proveedor de IA y tu plan de suscripción.

= ¿Qué tan seguro es el plugin? =
El plugin sigue las mejores prácticas de seguridad de WordPress con sanitización adecuada, verificación de nonce y almacenamiento seguro de claves API.
